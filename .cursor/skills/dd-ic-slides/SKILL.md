# DD 投資委員会資料の骨格更新（/dd:ic_slide）

**案件ルート**内の `review_and_hypothesis/投資委員会資料作成/スライドスケルトン.md` を、同案件の `dd materials/` と `review_and_hypothesis/` の情報で更新する。

## コマンド・プロンプト

- **コマンド**: `/dd:ic_slide`（対象は指定した案件ルート）
- **実行**: このスキルの手順・作業内容に従う

## 対象ファイル（案件ルート相対）

- **更新先**: 案件ルート内の `review_and_hypothesis/投資委員会資料作成/スライドスケルトン.md`

## 入力

- **投資委員会で強調したい論点**: 特に強調したい論点
- **結論の方向性**: 投資判断の結論の方向性

## 作業内容

1. **参照資料の確認**: 当該案件の `dd materials/` と `review_and_hypothesis/` の主要な資料を確認する
2. **スライドスケルトンの確認**: 既存のスライド構成（表紙・エグゼクティブサマリー・企業概要・事業概要等）を確認する
3. **更新**: 各スライドの「参照ファイル」を実際のパスに合わせて明記する。スケルトン内のパスは当該案件の `dd materials/` や `review_and_hypothesis/` の実フォルダ構成に合わせる
4. **論点・結論の反映**: 投資委員会で強調したい論点と結論の方向性を、該当スライドの記載内容に反映する

## 成果物

- 案件ルート内の `review_and_hypothesis/投資委員会資料作成/スライドスケルトン.md` が更新されている
- 各スライドで参照すべきファイルが正しいパスで明記されている
- 強調論点と結論の方向性が反映されている
