# DD サマリ作成（ステップ2.5）

コピーしたファイルのうち、Markdown 以外（PDF、Excel、Word、画像等）のサマリを、各フォルダ内の `{フォルダ名}サマリ.md` に記載する。

## コマンド・プロンプト

- **コマンド**: `/dd:summarize`
- **実行**: このスキルの手順・記載内容に従う

## 対象（いずれも案件ルート相対）

- **対象フォルダ**: 案件ルート内の `dd materials/business/`, `contract/`, `corporate/`, `financial/`, `round table/` 配下の各フォルダ
- **対象ファイル**: 各フォルダ内の .md 以外（.pdf, .xlsx, .docx, 画像等）
- **サマリファイル**: 各フォルダ内の `{フォルダ名}サマリ.md`（なければ新規作成）

## サマリに記載する内容（1ファイルあたり）

- **基本情報**: ファイル名、形式、サイズ、更新日時
- **内容サマリ**: ファイルの内容要約
- **主要なポイント**: 箇条書き
- **投資判断への影響**: このファイルが投資判断に与える影響

ファイル形式（PDF、Excel、Word等）に応じて適切な情報を抽出して記載する。

## 品質基準（dd-common 準拠）

サマリには「基本情報 / 内容サマリ / 主要ポイント / 投資判断への影響」が揃っていること。
