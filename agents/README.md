# agents/

このフォルダは、**Cursor（AIエージェント）にこのリポジトリでどう作業してもらうか**を整理するための指示書置き場です。

- `cursor/`: Cursor向けのシステム指示・擬似コマンド・チェックリスト
- `prompts/`: 各アクションを実行するための個別プロンプト（用途別）

推奨の入口は以下です。

- **擬似コマンド一覧**: `agents/cursor/commands.md`
- **このリポジトリでの作業ルール**: `agents/cursor/system.md`
- **DD方法（分割版）**: `agents/cursor/dd_method/00_index.md`
