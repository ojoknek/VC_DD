# /dd:wrap - 完了報告

## 目的
チェックリストで完了状態を確認し、次のステップ（追加ヒアリング/追加調査）を明確化します。

## 入力パラメータ
- **現時点の進捗**: 現在の作業進捗状況
- **未解決の論点**: まだ解決していない論点

## 完了確認チェックリスト

以下のチェックリストで完了状態を確認します：

- [ ] ステップ2: 業務整理用資料の分類・配置が完了
- [ ] ステップ2.5: 各フォルダ内のファイルサマリが作成・更新済み
- [ ] ステップ3: 事業概要.mdと投資条件.mdが更新済み
- [ ] ステップ4: ヒアリングリスト.mdが更新済み
- [ ] ステップ5: `dd materials`と`research`を素材にした初期レビューが完了し、`review_and_hypothesis`内の各評価ファイルが作成・更新済み
- [ ] ステップ6: review_and_hypothesis内の投資仮説ファイルが更新済み
- [ ] ステップ7: research内のリサーチ結果がまとめられている
- [ ] ステップ8: 参考資料が適切に保存されている

## 実行手順

1. **完了状態の確認**
   - 上記のチェックリストで完了状態を確認
   - 各ステップの完了状況を記録

2. **主要な発見事項の整理**
   - 重要な評価結果を整理
   - 主要な投資仮説を整理
   - 重要なリスク要因を整理

3. **次のステップの明確化**
   - 追加で必要な作業を特定
   - ヒアリングで確認すべき事項を明確化
   - 追加調査が必要なテーマを特定

4. **完了報告の作成**
   - 作業完了の確認
   - 作成・更新したファイルの一覧
   - 主要な発見事項
   - 次のステップ

## 成果物
- 完了報告が作成されている
- 次のステップが明確化されている

## 確認事項
- すべてのステップが完了しているか
- 重要な発見事項が漏れていないか
- 次のステップが明確になっているか
