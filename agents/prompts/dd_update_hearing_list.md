# /dd:questions - ヒアリングリスト更新

## 目的
事業概要と投資条件の更新結果を元に、質問を具体化・重複排除・優先度付けします。

## 入力パラメータ
- **既に把握している前提**: 事業概要や投資条件から把握できている情報
- **意思決定で重要な論点**: 投資判断に必要な重要な論点

## 対象ファイル
- `review_and_hypothesis/dd question/ヒアリングリスト.md`

## 実行手順

1. **既存情報の確認**
   - `dd materials/business/事業概要.md` を確認
   - `dd materials/round table/投資条件概要.md` を確認
   - 既に把握している情報をリストアップ

2. **不足情報の特定**
   - 投資判断に必要な情報で、まだ把握できていない情報を特定
   - 優先度の高い質問項目を明確化

3. **ヒアリングリストの更新**
   - 既に把握している情報を反映
   - 不足している情報を質問として追加
   - 優先度の高い質問項目を明確化
   - 事業概要や投資条件に関連する質問を追加・調整

4. **質問の整理**
   - 重複する質問がないか確認
   - 質問の具体性を確認
   - 回答可能な形式になっているか確認

## 成果物
- `review_and_hypothesis/dd question/ヒアリングリスト.md` が更新されている
- 既に把握している情報が反映されている
- 不足している情報が質問として追加されている
- 優先度の高い質問項目が明確化されている

## 確認事項
- 重複する質問がないか確認
- 質問の具体性を確認
- 回答可能な形式になっているか確認
- 優先度が適切に設定されているか
