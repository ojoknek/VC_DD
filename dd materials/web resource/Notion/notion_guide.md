# Notion MCP ガイド

Notion MCPを使用してNotionのデータベースにアクセスする方法です。

## データベースへのアクセス方法

### 1. データベースIDを取得する

NotionのデータベースページのURLからデータベースIDを取得します。

**URL形式：**
```
https://www.notion.so/[ワークスペース名]/[データベースID]?v=...
```

**例：**
```
URL: https://www.notion.so/myworkspace/12345678-1234-1234-1234-123456789abc?v=...
データベースID: 12345678-1234-1234-1234-123456789abc
```

### 2. データベースを検索する

データベース名やページタイトルで検索してアクセスします。

**使用例：**
- 「投資案件管理」で検索
- 「DD資料」で検索

### 3. データベースをクエリする

データベースIDを指定して、データベース内のページ（レコード）を取得します。

**主な操作：**
- フィルタ条件を指定してデータを絞り込み
- ソート条件を指定して並び替え
- ページネーションで全件取得

### 4. ページ情報を取得する

ページIDを指定して、ページの詳細情報やプロパティを取得します。

---

## 基本的な使い方

1. **データベースを検索** → データベースIDを取得
2. **データベースをクエリ** → 条件に合致するデータを取得
3. **ページ情報を取得** → 詳細情報を確認

---

## 注意事項

- データベースがボットに適切に共有されている必要があります
- データベースIDは32文字の英数字（ハイフンを含む）です
- プロパティ名は正確に入力してください
