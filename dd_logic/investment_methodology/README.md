# 投資思考法（共通テンプレート）

全案件（`deals/*`）の `review_and_hypothesis` において**通底する**、投資判断にかかる思考法を記述するためのテンプレート群です。  
ここに置かれた内容は**人間が記述・編集**し、案件ごとの投資テーゼやレビューで参照します。

## 位置づけ

- **dd_logic/investment_methodology/** … 全案件に共通する「どう考えるか」のフレームワーク（ここで編集）
- **各案件の review_and_hypothesis/senario/investment_theory.md** … 当該案件の投資テーゼ・シナリオ（共通思考法を参照しつつ案件ごとに記述）

## ファイル一覧

| ファイル | 用途（人間が記入する内容） |
|----------|----------------------------|
| [投資の考え方.md](投資の考え方.md) | なぜ投資するか、何を重視するか、投資方針 |
| [評価の観点.md](評価の観点.md) | 事業・財務・組織・リスク等の評価観点・チェックリスト |
| [リスクの見方.md](リスクの見方.md) | リスクの分類・見方・許容基準 |
| [出口の考え方.md](出口の考え方.md) | 出口戦略の考え方・評価の観点 |

## 使い方

1. 上記 md を開き、自組織のスタンスや基準に合わせて**人間が記述・更新**する。
2. 各案件で `/dd:thesis` やレビューを行う際、エージェント・人間の双方がこのフォルダを参照し、案件ごとの `review_and_hypothesis` に反映する。
