# deals（案件フォルダ）

DD Template において、**1件ごとのデューデリジェンス案件**を格納するためのルートです。

## 役割

- **案件の格納**: 会社ごとに `deals/{会社名}/` のようなサブフォルダを切り、その中に当該案件の資料・サマリ・レビュー・投資仮説などを置く
- **案件ルート**: 擬似コマンド（`/dd:copy`・`/dd:review` など）はすべて **「案件は deals/株式会社ABC」** のように、この `deals/` 直下のいずれか1フォルダを「案件ルート」として指定して実行する
- **テンプレートのコピー元**: 新規案件は `dd_logic/template` を `deals/{会社名}/` にコピーして作成する

## 注意（Git）

- **`deals/` 直下のファイル・フォルダは .gitignore で無視**されています（機密・案件別データをリポジトリに含めないため）
- この README は `deals` 自体の説明用で、リポジトリに含めたい場合は `deals/` のみ ignore せず `deals/**` で下層だけを ignore する設定になっています

## 新規案件の作り方

```bash
cp -r dd_logic/template deals/株式会社ABC
```

その後、`業務整理用/` に資料を配置し、`/dd:copy` や `/dd:init` などで進めます。手順の詳細はプロジェクトルートの [README.md](../README.md) を参照してください。
